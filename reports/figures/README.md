# Visualization Figures

This directory contains all generated visualizations from the disaster prediction analysis.

## Directory Structure

```
figures/
├── eda/              # Exploratory Data Analysis visualizations
│   ├── country_distribution.png
│   ├── continent_analysis.png
│   ├── disaster_types.png
│   ├── correlation_heatmap.png
│   ├── temporal_trends.png
│   └── ... (50+ EDA visualizations)
│
└── results/          # Model results and performance visualizations
    ├── confusion_matrix.png
    ├── roc_curves.png
    ├── feature_importance.png
    ├── model_comparison.png
    ├── predicted_vs_actual.png
    └── ... (40+ result visualizations)
```

## Visualization Categories

### 1. Frequency Analysis
- Disasters by country
- Disasters by continent
- Disasters by type
- Time series plots

### 2. Geographic Visualizations
- World maps with disaster locations
- Heatmaps of hotspots
- Regional distribution

### 3. Economic Impact
- Economic loss by country/continent
- Top 10 most costly disasters
- Aid distribution

### 4. Response Analysis
- Response efficiency scores
- Response time analysis
- Effectiveness metrics

### 5. Recovery Analysis
- Recovery days by disaster type
- Recovery vs severity
- Efficiency comparisons

### 6. Model Performance
- Confusion matrices
- ROC curves
- Feature importance
- Model comparisons

### 7. Correlation Analysis
- Feature correlation heatmaps
- Relationship plots

### 8. Temporal Trends
- Yearly trends (2018-2024)
- Monthly patterns
- Seasonal variations

---

**Note:** All figures are automatically generated by running the Jupyter notebooks in sequential order (Phase 2 → Phase 5).
